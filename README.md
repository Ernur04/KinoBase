# KinoBase
KinoBase — большая база фильмов, сериалов, аниме и мультфильмов.

## Описание проекта
KinoBase представляет собой современный веб-интерфейс для каталога видеоконтента. Проект включает в себя адаптивный дизайн, поддержку светлой и темной темы, а также удобную навигацию по разделам сайта.

## Технологии
- **HTML5** (Семантическая разметка)
- **CSS3** (Flexbox, Grid, CSS Variables для тем)
- **Vanilla JavaScript** (Логика слайдеров, переключение тем)
- **FontAwesome** (Иконки)
- **Google Fonts** (Шрифт Montserrat)

## Основные фичи
- 🌓 **Переключение темы** (Светлая / Темная)
- 📱 **Адаптивный дизайн** (Grid сетки для карточек фильмов)
- 🎬 **Слайдер новинок** (Интерактивный баннер)
- 🧭 **Удобная навигация** (Выпадающее меню жанров)

## Запуск проекта
Проект состоит из статических файлов. Для запуска достаточно открыть `index.html` в любом современном браузере. Для комфортной разработки рекомендуется использовать расширение **Live Server** для VS Code.
